## Learning Yacc/Bison
_Stop yacking, start yaccing!_

### What is what
    
    parse.y         : define the grammar
    parse.tab.c     : yacc generated grammar parser
    lex.l           : token definitions
    lex.yy.c        : lex generated token parser
    tokens.h        : token symbols


### How to
```bash
$ make
$ ./a.out
```
